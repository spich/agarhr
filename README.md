# agarhr

Open-source agar.io klon s modernim Node.js 20+ serverom i React frontendom.

## Struktura
- `server/` — WebSocket game server (Node.js 20+)
- `client/` — React front (canvas rendering)

## Pokretanje

### Server
```bash
cd server
npm install
npm start
```

### Client
```bash
cd client
npm install
npm start
```
